# 災害が科学に変わった瞬間　〜1755年リスボン大震災を境に科学はどう変わったか？〜

1755年11月1日、ポルトガルの首都リスボンを襲った大地震。M8.5〜9の揺れ、津波、火災の三重災害は、ヨーロッパ全域に衝撃を与えました。

「なぜ神は善良な市民の街を破壊したのか？」

この問いに対し、
・ カントは「地震は神罰ではなく自然現象」と主張
・ ミッチェルは地震波の伝播理論を提唱
・ そしてマレットはミッチェルの理論を実験で検証し、「地震学」という言葉を生み出しました。

災害が科学に変わった瞬間——それは「なぜ」から「どのように」への問いの転換でした。

🔍 この動画でわかること：
・ 1755年リスボン大震災の被害と衝撃
・ 科学革命前夜の「観察の文化」と英国王立協会の役割
・ カント、ミッチェル、マレットによる地震学の確立
・ 神学的解釈から科学的探求への歴史的転換
